## 页面结构

### 🚪 入口层 (Entry Overlay)
- 黑底 `#1a1a1a` 全屏遮罩，`z-index: 1000`
- 居中白色卡片，带浮动动画 `entryFloat`
- 输入框 + 确认按钮，支持回车提交
- 正确答案：`张婷婷`
- 错误提示：8 条随机搞笑文案，卡片抖动 + 输入框变红
- 正确：独立礼花 canvas (`#entry-firework-canvas`，`z-index: 1001`），4 波礼花后遮罩淡出

### 🎵 音乐按钮
- 右上角固定，`z-index: 200`
- 点击播放/停止，使用 Web Audio API 合成生日快乐歌
- 播放中带脉冲动画 `musicPulse`

### 🌸 樱花画布
- 全屏 canvas `#sakura-canvas`，`z-index: 100`
- 30 片花瓣飘落，粉/白随机

### 📝 祝福卡片
- 毛玻璃卡片 `.card`（`backdrop-filter: blur(10px)`）
- IntersectionObserver 触发逐行淡入
- 10 行祝福语，每行延迟 250ms

### 📿 许愿滑动条 (Wishing Slider)
- 位置：蛋糕**上方**
- 横向轨道 + 圆形滑块（🕯️），`touch-action: none`
- 支持鼠标拖拽 + 触摸滑动 + 点击轨道快进
- 滑到 93% 触发完成：滑块变 ✨，提示文字变色
- 自动触发音乐播放
- 10 秒后弹出提示 "🕯️ 现在可以点击熄灭蜡烛啦！" + 叮咚音效
- **无 CSS transition**，纯 JS 实时跟手

### 🎂 蛋糕 (Cake)
- 结构（从上到下）：
  ```
  蜡烛 × 3（火焰 + 灯芯）
  ┌──────────────┐  ← 顶层 128px (最浅粉)
  ├──────────────┤  ← 奶油裱花 140px (白色)
  ├──────────────┤  ← 中层 164px (中等粉)
  ├──────────────┤  ← 奶油裱花 176px
  ├──────────────┤  ← 底层 200px (最深粉)
  ├──────────────┤  ← 奶油裱花 210px
  ════════════════  ← 蛋糕托盘 220px (金色椭圆)
  ```
- 布局方式：`.cake` 固定 `width: 220px; margin: 0 auto`（纯 block 居中，不用 flex）
- 每层使用横向渐变（90deg）模拟 3D 圆柱光照
- 火焰动画 `flicker` 0.4s 交替，3 根蜡烛不同延迟
- 无樱桃、无糖粒、无甘纳许伪元素

### 🍀 好运签 (Fortune Cards)
- 6 张 2×3 网格翻转卡片
- CSS 3D transform `rotateY(180deg)`
- `onclick="this.classList.toggle('flipped')"`

### 📢 蜡烛弹窗 (Candle Popup)
- `position: fixed; bottom: 100px`
- 粉红渐变背景，白色文字
- `popupSlideIn` 动画进入，`popupSlideOut` 退出
- 6 秒后自动消失

## JavaScript 模块

| 模块 | 功能 | 关键点 |
|------|------|--------|
| Entry Screen | 名字验证 + 入口礼花 | 正确名字硬编码 `'张婷婷'`，独立 canvas 粒子系统 |
| Sakura Petals | 樱花飘落 | Canvas 2D，30 片，requestAnimationFrame |
| Fireworks | 烟花粒子 | 独立 canvas，60 粒子/次，重力 + 衰减 |
| Candle Blowing | 吹蜡烛 + 祝福字幕 | 首次：吹灭+烟花+4s重燃。之后：点蛋糕飘随机祝福(30条) |
| Wishing Slider | 滑动许愿 | mouse/touch 事件，无 CSS 过渡 |
| Scroll Reveal | 滚动显示 | IntersectionObserver，threshold 0.2 |
| Music Box | 生日快乐歌 | Web Audio API 合成，振荡器+vibrato，tempo=0.2s/音符 |
| Click Particles | 点击粒子 | 10 个随机 emoji 散开，0.7s 飞出+淡出 |
| Ding Sound | 提示音效 | 两段正弦波 880Hz+1100Hz |

### 音乐参数
```
noteDuration = 0.2s（每个音符）
音符间隔 = 0.05s
乐句停顿 = 0.15s
长音符倍数 = 1.6x
音调 = C调（392Hz ~ 784Hz）
```

## 已知问题 / 注意事项

1. **GitHub 连接不稳定**：国内偶尔被墙，Gitee 可正常推送，需手动同步
2. **音频自动播放限制**：部分浏览器需用户先交互（滑动许愿会触发点击 music 按钮，可绕过）
3. **单文件**：所有 CSS/JS 内联在 `index.html`，修改时注意行号偏移
4. **蛋糕布局**：用 `margin: 0 auto` 居中，不要改 flex，不要加 `position: absolute` 伪元素
5. **滑动条**：不要加 CSS transition，跟手靠 JS 实时赋值 `left` / `width`
